Welcome to Qooxdoo-Maven !

Qooxdoo-Maven aims to simplify the development of Qooxdoo applications by using Java technologies.

Builds

Important: if the link points to an empty page, make sure you have builded the application/component by using the appropriate maven command !

Production

  • Launch 
    mvn compile
    to build the production version of your application
  • Launch 
    mvn qooxdoo:testrunner
    to build the testrunner application (unit tests)

Development

  • Launch 
    mvn -Dbuild.type=dev compile
    to build the development version of your application
  • Launch 
    mvn -Dbuild.type=dev qooxdoo:testrunner
    to build the testrunner application (unit tests)

Configuration

  • Application namespace : ${qooxdoo.application.namespace}
  • Path to application class dir : ${qooxdoo.application.sourcesDirectory}
  • Path to application resource dir : ${qooxdoo.application.resourcesDirectory}/${qooxdoo.application.namespace}
  • Path to application translation dir" : ${qooxdoo.application.translationDirectory}
  • Qooxdoo path : ${qooxdoo.sdk.parentDirectory}/qooxdoo-sdk
  • Qooxdoo Theme : ${APPLICATION}.theme.Theme
  • Qooxdoo root : ${qooxdoo.application.outputDirectory}/${qooxdoo.application.namespace}

Have a look to config.json and Manifest.json for a details view of the configuration.

To change permanently the application configuration, you must change the files located into the source directory (
${qooxdoo.application.resourcesDirectory}/${qooxdoo.application.namespace}/config
)
